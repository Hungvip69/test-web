About the Project

Empowering the Visually Impaired Through Innovation

The Artificial Eye project was born from a desire to create a comprehensive solution that addresses multiple challenges faced by blind individuals in their daily lives. Unlike single-function assistive devices, our system integrates seven essential modes into one intuitive platform.

By combining advanced software capabilities with custom 3D-printed hardware, we've created a device that's both highly functional and affordable. Our goal is to bridge the accessibility gap and provide blind individuals with greater independence.

Winner of the 2023 National Science Fair Innovation Award

Key Features

AI-Powered Assistance

Advanced machine learning algorithms provide accurate object recognition, text-to-speech conversion, and intelligent responses to user queries.

7 Integrated Modes

Comprehensive functionality including movement assistance, reading, chatbot interaction, time telling, weather updates, rest mode, and distance measurement.

3D-Printed Hardware

Custom-designed, lightweight housing that's affordable to produce and can be easily customized for individual user needs.

Voice Control

Natural language processing allows for intuitive voice commands and conversational interaction with the device.

Smartphone Integration

Companion app allows caregivers to monitor usage, adjust settings, and receive alerts when assistance might be needed.

Seven Essential Modes

Movement Assistance

The device uses a camera and AI to find obstacles (like people or objects) and tells the user how far they are by using an ultrasonic sensor connected to an Arduino board. The AI also says if it’s a person or just an object. This helps blind people walk safely and avoid things in their way.

Reading Mode

The camera takes a picture of text. The device uses AI to read the text and lets the user choose to hear a summary or the full content. It then reads it out loud. This helps users get information quickly and easily.

Chat Assistant

Listens to the user, sends the question to ChatGPT, and reads out the answer.

Time Mode

Says the current time and reminds about special events.

Weather Updates

Tells the current weather and gives advice about going outside.

Rest Mode

The device uses AI and a camera to recognize faces. It tells the user if people nearby are family members or strangers, so the user feels safer while resting.

Distance Measurement

Uses the Arduino and ultrasonic sensor to measure and tell the user how far away obstacles are.

3D-Printed Hardware

Custom-Designed for Distance Mode

Our 3D-printed housing was specifically engineered to optimize the distance measurement capabilities of the Artificial Eye system. The lightweight yet durable design incorporates:

  • Precision mounting for ultrasonic sensors
  • Ergonomic shape for comfortable wear
  • Modular design for easy upgrades
  • Vibration motors for haptic feedback

The housing can be 3D printed in about 8 hours using standard PLA filament, keeping production costs under $5 per unit.
